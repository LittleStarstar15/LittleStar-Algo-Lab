MAIN TERMINAL

量子星图网络：图 (GRAPH)

THE ULTIMATE MANY-TO-MANY DATA NETWORK

挣脱层级枷锁的终极形态：万物互联。

如果二叉树是严格的“上下级组织架构”，那么 **图 (Graph)** 就是错综复杂的“人类社交网络”或“宇宙星图”。它没有固定的起点，没有任何层级限制，任何人都可以和任何人建立连接。

它由 节点 (Vertex / 星系) 和 边 (Edge / 跃迁航线) 组成。你手机里的高德地图导航底层、微信好友推荐、互联网的路由协议，全都是运行在这种结构之上的复杂算法！

▦ 邻接矩阵 (Adjacency Matrix)

用一个 N×N 的二维数组来记录。
如果节点 A 到 B 有连线，则 `Matrix[A][B] = 1`。极其消耗空间，但查询连通性是 O(1)。

Add nodes in the Engine view first.

🔗 邻接表 (Adjacency List)

用一个数组，里面嵌套多条链表。
只记录真正存在的连线。极其节省空间，是真实工程中最主流的存储方式。
