星际数据收容单元：数组 (ARRAY)

CONTIGUOUS MEMORY ALLOCATION & O(1) ACCESS

计算机科学中最伟大的发明：连续内存。

数组（Array）之所以是所有编程语言的基石，是因为它完美契合了物理内存的硬件特性：连续分配。
这就好比你在太空港包下了一整排连在一起的飞船停机位。只要知道第一个机位的编号（基地址 Base Address），再知道每个机位的宽度（元素大小 Element Size），AI 就能通过一行数学公式，在 0.00001 毫秒内瞬间锁定第 1000 个机位的确切位置。这就是恐怖的 O(1) 随机访问。
